I developed the website for CDT (Corporación de Desarrollo Tecnológico), a branch of the Cámara Chilena de la Construcción focused on driving innovation, productivity, and sustainability in the construction industry. The platform serves as a central hub for news, technical content, multimedia resources, normative updates, and events such as ENAMAC, ENASOC, and ENAPROD.

The front-end is built with Astro and React components, delivering a fast, modern experience. The back-end runs on WordPress, managing multiple custom post types that cover different content categories like news, technical documents, podcasts, and streaming. I implemented a likes system that allows users to save and interact with content, along with a full authentication flow for both regular users and site administrators. The private zone lets each user define their content preferences, creating a personalized feed tailored to their interests within the construction sector.
