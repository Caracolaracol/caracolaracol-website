## Farmacias Ahumada - Landing Club Bebe

A comprehensive landing page and membership system developed for Farmacias Ahumada's exclusive Club Bebe program. This project focused on creating an engaging entry point for parents to join a benefits club designed specifically for baby care products and services.

### Key Features

**Landing Page Design**
- Custom-designed landing page showcasing Club Bebe benefits and exclusive offers
- Responsive design optimized for both desktop and mobile experiences
- Integration with Farmacias Ahumada's brand identity and e-commerce ecosystem

**Membership Registration System**
- Interactive inscription form allowing users to register for club membership
- Custom validation and user experience optimization
- Seamless integration with Salesforce Commerce Cloud infrastructure

**Technical Integration**
- Frontend development using HTML, CSS, and JavaScript
- Backend integration with WordPress for content management
- Salesforce Commerce Cloud integration for e-commerce functionality
- Form data processing and member management system

### Technical Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: WordPress
- **E-commerce Platform**: Salesforce Commerce Cloud
- **Integration**: Custom APIs and form handling

### Project Impact

This landing page served as the primary conversion point for Club Bebe membership, providing parents with easy access to exclusive benefits, discounts, and specialized baby care products through Farmacias Ahumada's e-commerce platform. The project successfully bridged the gap between marketing initiatives and technical implementation, creating a smooth user journey from awareness to membership registration.
