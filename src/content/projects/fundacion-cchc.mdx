## Fundación CChC

A modern, content-driven website developed for Fundación CChC (Fundación de la Cámara Chilena de la Construcción). This project focused on creating a robust platform to showcase the foundation's social initiatives, educational programs, and community impact in Chile's construction sector.

### Project Overview

The website serves as a digital hub for Fundación CChC's mission to promote social development, education, and community welfare within Chile's construction industry. Built with modern web technologies, the platform provides an efficient content management system for the foundation's team while delivering an optimal user experience for visitors seeking information about programs, initiatives, and ways to get involved.

### Key Features

**Content Management System**
- Powered by Strapi CMS for flexible content creation and management
- Custom content types tailored to foundation activities and programs
- User-friendly admin interface for non-technical staff
- Multi-media support for showcasing projects and impact stories

**Modern Frontend Architecture**
- Built with Astro.js for optimal performance and SEO
- Static site generation with dynamic content integration
- Component-based architecture for maintainable code
- Responsive design across all device types
- Some Client side components are built with React

**Foundation-Specific Features**
- Program showcase sections highlighting educational initiatives
- Impact stories and testimonials from beneficiaries
- News and updates system for foundation activities
- Contact and collaboration forms for community engagement
